Установка клиентского терминала MetaTrader (шаг 4)

На экране окно, в котором Вам необходимо выбрать путь, по которому будет установлен программа.

Примечание: Под одной операционной системой Microsoft Windows можно установить 
несколько клиентских терминалов MetaTrader, для каждого используя отдельный 
путь установки и отдельную группу (группа в меню "Пуск - > Программы")!

Для продолжения установки нажмите кнопку "Далее >".
